my ch embl is a completely open platform, which combines public domain bioactivity data with open source database and cheminformatics technologies. my ch embl consists of a Linux (Ubuntu) Virtual Machine featuring a PostgreSQL schema with the latest version of the ch embl database, as well as the latest rd kit cheminformatics libraries.
